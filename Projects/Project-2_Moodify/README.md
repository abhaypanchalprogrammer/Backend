# 🎵 Moodify – AI-Based Music Recommendation App

Moodify is a full-stack web application that detects a user's facial expressions in real-time and recommends songs based on their mood.

## 🚀 Live Demo
🔗 https://moodify-x9bt.onrender.com/

## 📌 Features
- 🎥 Real-time facial expression detection using MediaPipe  
- 😊 Detects moods like Happy, Sad, Surprised, etc.  
- 🎶 Dynamic music recommendations based on mood  
- 🔐 Secure authentication using JWT & bcrypt  
- 📱 Fully responsive UI built with React  

## 🛠 Tech Stack

### Frontend
- React.js  
- Tailwind CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB (Mongoose)  

### Other Tools
- MediaPipe (Face Detection)  
- JWT Authentication  
- ImageKit (if used)  

## ⚙️ Installation & Setup

1. Clone the repository:
```
git clone https://github.com/abhaypanchalprogrammer/Backend.git

Navigate to the project folder:
cd backend/projects/Project-2_Moodify

Install dependencies:
npm install

Create a .env file and add:
PORT=3001
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

Run the server:
npm start
```
